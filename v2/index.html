Privacy

This page has no backend. There is no server collecting your data, because there is no server at all — it's a single static HTML file, viewable by anyone via "view source" or this repository's public code on GitHub.

Every request this page makes goes directly from your browser to Octopus Energy's own API at api.octopus.energy, using your own API key. Nothing you enter is sent to the author of this tool, to any analytics service, or to any third party of any kind. You can verify this yourself by opening your browser's network tab while using the page.

The one exception is the "Email this summary to auth.energy" button on the results screen, which is entirely optional. It opens a draft in your own email app — it doesn't send anything automatically — containing only your postcode's outward code (e.g. "N15", never the full postcode), the comparison period, total kWh, and the actual/Flexible/Agile totals. It never includes your API key, account number, MPAN, meter serial, or raw consumption data, and you can review or cancel the draft before anything leaves your device.

If you tick "remember my API key," only your API key is saved using your browser's local storage — a feature built into the browser itself, not a cookie, and not transmitted anywhere. Your account number, MPAN, and meter serial are discovered from Octopus each time and are not stored. The saved key stays on your device until you clear it (via the button on the input screen, or by clearing your browser's site data for this page) and is never readable by any other website.

Treat your Octopus API key like a password. Anyone with it can read your account's billing and consumption history. If you use the "remember" option, that key sits in plain form in this browser's storage, so avoid using it on a shared or public computer, and clear it when you're done if you do.
